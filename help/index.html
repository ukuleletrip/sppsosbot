SPP SOS LINE Bot

Demonstration of SOS API by SPP

基本的な使い方

センサ値を確認する

以下のようなメッセージを送るとセンサ値を確認することができます。

  • 気温は？
  • temperature?
  • 湿度は？
  • humidity?
  • 風速は？
  • wind speed?
  • 気圧は？
  • pressure?
  • 降雨は？
  • precipitation?
  • 日射は？
  • irradiance?
  • solar?

アラートを設定する

以下のような形式のメッセージを送るとアラートを設定することができます。

[センサ名]が[センサ値][条件][知らせて|教えて]

  • 気温が10.5度以上になったら知らせて
  • 湿度が50%より小さくなったら知らせて
  • 雨量が0.5mmより大きくなったら教えて
  • 風速が5m以下になったら知らせて
  • 温度が30℃より高くなったら教えて

指定した条件に達した時、また対象データが途絶している時に、メッセージで通知します。
アラートはユーザ毎に1つだけ設定可能です。新たにアラートを設定すると前のアラートは上書きされます。
